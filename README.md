# intelligent-air-chair
# 智能气垫椅 / 飞机座椅温控系统

> 基于 ESP32 触摸屏的座椅加热温控系统，支持可视化温度显示与触摸滑块调档。
> 面向飞机/汽车座椅场景，纯白航空风界面，电热片三段控温 + 一键关闭。

## ✨ 功能特性

- 🌡️ **实时温度显示** — DS18B20 采集座椅温度，大字实时刷新
- 🎚️ **触摸滑块调档** — 拖动滑块四档切换：`OFF / 30°C / 34°C / 38°C`，松手自动吸附
- 🔥 **三段控温** — 低于目标全功率加热，接近目标平滑收敛，到温停止
- ⏱️ **高温自动降档** — 38°C 高温档运行 15 分钟后自动降到 34°C，界面实时倒计时
- 💡 **板载 RGB 档位灯** — 蓝(30)/绿(34)/红(38)/灭(OFF)，不看屏也知道档位
- 🛡️ **安全默认** — 上电默认 OFF，不主动加热
- 🎨 **航空高级风 UI** — 纯白底 + 深海军蓝字 + 圆角滑块 + 矢量字体

## 🧰 硬件

| 部件 | 型号/规格 |
|------|----------|
| 主控带屏板 | Sunton **ESP32-3248S035R**（3.5" ST7796U 320×480 + XPT2046 电阻触摸 + ESP32-WROOM-32E） |
| 温度传感器 | DS18B20（+ 4.7kΩ 上拉） |
| 加热驱动 | MOS 驱动模块 ×N |
| 加热元件 | 电热片（独立 DC 适配器供电） |

## 🔌 接线速览

| 设备 | 接到 ESP32 |
|------|-----------|
| DS18B20 DATA | **IO25**（VCC→3.3V，DATA↔3.3V 间接 4.7kΩ） |
| MOS 信号 IO-PWM | **IO32**（多路并联） |
| 公共地 | ESP32 GND、适配器 GND、MOS GND 全部共地 |

> ⚠️ 加热片用独立适配器供电，**不可**从 ESP32 取电；3.3V 与适配器正极**绝不相连**。
> 完整接线图见 [docs/接线图.md](docs/接线图.md)。

## 🚀 快速开始

1. Arduino IDE 安装 **ESP32** 开发板支持，选 `ESP32 Dev Module`
2. 安装库：**LovyanGFX**、**OneWire**、**DallasTemperature**
3. 先烧 `code/00_screen_test` 验证屏幕，再烧 `code/03_touchscreen_ui` 正式固件
4. 上传速度建议 **115200**（CH340 在 921600 易握手失败）

详细步骤见 [docs/实施文档.md](docs/实施文档.md)。
