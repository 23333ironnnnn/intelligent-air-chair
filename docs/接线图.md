## 2. ESP32 引出口(I2C 4pin 座子)

板子上唯一空闲可用的引脚都在这个座子上,丝印顺序:

```
┌──────── I2C 座子 (4pin) ────────┐
│  3.3V  │  IO32  │  IO25  │  GND  │
└────┬───────┬────────┬───────┬───┘
     │       │        │       │
   传感器   加热     传感器    公共
   供电+   信号     DATA      地
```

> ⚠️ `IO35 / IO39` 那个座子是**只能输入**的脚,不能驱动加热,别用。

## 4. 引脚对照表

| 设备 | 引脚/端子 | 接到 | 备注 |
|------|----------|------|------|
| DS18B20 | VCC | ESP32 **3.3V** | ⚠️ 不是 5V |
| DS18B20 | DATA | ESP32 **IO25** | |
| DS18B20 | GND | 公共地 | |
| DS18B20 | 4.7kΩ | DATA ↔ 3.3V 之间 | 上拉,裸元件必须加 |
| MOS ×N | IO-PWM | ESP32 **IO32**(并联) | 4 路信号合一,等效 |
| MOS ×N | GND | 公共地 | 信号共地,必须 |
| MOS ×N | DC+ / DC- | DC 适配器 | 加热独立供电 |
| MOS ×N | OUT+ / OUT- | 电热片 | |
| DC 适配器 | GND | 公共地 | 与 ESP32 共地 |

> 固件里:`PIN_DS18B20 = 25`,`PIN_HEATER = 32`(4 路 MOS 始终同一 PWM,故合并到一脚)。

---

## 5. ⚠️ 三条安全铁律

1. **共地**:ESP32 GND、DC 适配器 GND、所有 MOS 的 GND —— **全部连在一起**。不共地则 PWM 信号无效。
2. **正极隔离**:ESP32 的 **3.3V** 和 加热适配器的 **正极**(DC+)**绝不能相连**。一旦短接,高压灌入 ESP 会烧板。
3. **传感器用 3.3V**:DS18B20 供电和上拉都接 **3.3V**,不能用 5V(否则 5V 经 DATA 顶坏 IO25)。

---

## 6. 加热独立供电说明

电热片很耗电,**不能**从 ESP32 取电。它由独立 DC 适配器供电,ESP32 只通过 IO32 给 MOS 发"开/关"小信号,MOS 像开关一样把适配器的电放给电热片:

```
ESP IO32 (3.3V 小信号) ──→ MOS 触发 ──→ 把适配器的电导通给电热片
                            ↑
                      DC 适配器(大电流)
```

---

## 7. 上电自检顺序(建议)

1. 先只接 DS18B20,上电看屏幕显示真实室温(非 -127)→ 传感器 OK
2. 再接 MOS + 电热片,滑块拖到 34/38,看状态行 `HEATING` + 电热片变热 → 加热 OK
3. 拖到 OFF,确认加热停止、RGB 灯灭
4. 全程盯着,电热片**烫手/焦味/冒烟立即断电**
